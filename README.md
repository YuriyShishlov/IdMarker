# Проект Windows Forms
## Бизнес-логика размещена в отдельном проекте
## Dependency injection в конструкторе

Идея родилась на работе.
При прошивке электронных плат с микропроцессором,
используется флешка с программным обеспечением.
Нужно было сделать копирование папки
с исходной флешки на резервную.
При этом нужно было изменить число ID, 
хранящееся в файле типа DIP230_ID.txt

# Требования к задаче

## Первая часть программы

1. Программа должна осуществлять копирование папки с файлами
с исходной флешки на резервную.
2. Должна быть возможность выбора папки с файлами на исходной флешке
в проводнике
3. Программа должна находится на резервной флешке
4. Программа должна сама определить путь, где она находится
5. На исходной флешке, в выбранной папке, программа должна найти файл
типа DIP230_ID.txt, в котором хранится число
6. Число, хранящееся в файле типа DIP230_ID.txt на исходной флешке
должно быть увеличено на число, заданное в соответствующем TexBox
(по умолчанию 100)
7. Число, хранящееся в файле типа DIP230_ID.txt на резервной флешке
должно быть сохранено без изменений

## Вторая часть программы

1. Должна быть возможность выбора файла на резервной флешке типа DIP230_ID.txt, 
в котором хранится число, которое копируется в служебный файл программы

2. По клику на кнопку "Сколько" программа вычисляет разницу чисел в выбранном файле
типа DIP230_ID.txt и служебном файле, результат выводится в TexBox
